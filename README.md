# School Information Mobile App
🚀 React Native Expo App

This is a **React Native** app built with **Expo**.  
Follow these steps to run it locally.

---

## 📦 Prerequisites

Make sure you have:

- **Node.js**
- **npm** (comes with Node.js)
- **Expo Go App** on your phone (iOS App Store / Google Play)

---

## 🚀 Getting Started

1. **Download this project** and open it in your terminal.

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```


## 📱 Running the App

Once the development server is running:

On your phone:

>Open the Expo Go app → Scan the QR code shown in your terminal or browser.


On an emulator/simulator:
While the development server is running:

>Press i in the terminal for iOS Simulator (Mac only)

>Press a in the terminal for Android Emulator



⚠️ Important: You must run npm start before scanning the QR code or pressing i / a.
